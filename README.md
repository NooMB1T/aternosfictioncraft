## 🔧 ЕСЛИ ЧТО-ТО СЛОМАЛОСЬ

### ❌ "Ошибка подключения к Aternos"
1. Проверь интернет соединение
2. Проверь логин **GodB1T** и пароль **AntonGod** правильные
3. Может быть, Aternos блокирует запросы - попробуй VPN
4. Перезагрузи Replit (Stop → Run)

### ❌ "Не вижу список игроков"
- Это нормально! Сервер загружается 5-10 минут после `/on`
- Подожди и попробуй снова командой `/players`

### ❌ "Бот не отвечает или завис"
1. Нажми **Stop** в Replit
2. Нажми **Run** - перезагрузит бота
3. Попробуй команду снова

### ❌ "Бот уснул (не отвечает с утра)"
- Это значит, что UptimeRobot не пингует сервер
- Проверь в настройках UptimeRobot:
  - Правильный ли URL указан?
  - Включен ли мониторинг?
  - Может быть, нужен VPN?

### ❌ "Много ошибок в консоли Replit"
- Скопируй текст ошибки
- Перезагрузи бот (Stop → Run)
- Если помогло - всё ок
- Если нет - проверь интернет и настройки
